# Norskvenn 安装 + 使用指南（傻瓜版）

给任何挪威语 YouTube 视频自动生成 **挪威语 + 英语** 双语字幕。专为语言学习者做的。

---

## 这是什么

一个 **Chrome 浏览器插件**。打开挪威语 YouTube 视频，点一个按钮，它就用 AI 把语音转成字幕，**挪威语 + 英语对照**显示在视频上。

有两种工作模式：

| 模式 | 花钱吗 | 准确度 | 速度 |
|---|---|---|---|
| **本地模式**（默认） | 免费 | 一般 | 慢（看电脑性能） |
| **OpenAI 模式** | 要（约 ¥2.5/小时视频） | 很准 | 快 |

新手建议：先用免费的本地模式试试。觉得不够准、愿意花点小钱，再切 OpenAI 模式。

# 第一部分：安装（5 分钟，一次性）

## 准备
- 一台 Mac 或 Windows 电脑
- **Chrome 浏览器**（没有的话 [点这里下载](https://www.google.com/chrome/)。Edge / Brave / Arc 也行）

## 第 1 步：下载
打开 👉 **https://github.com/lp-capri-pl/norskvenn/releases/latest**

往下滚到 **"Assets"**，点最新的 **`norskvenn-vX.X.X.zip`** 下载（X 是版本号，下最新的就行）。

## 第 2 步：解压
到"下载"文件夹找到刚下的 zip：
- **Mac**：双击，自动解压出一个文件夹
- **Windows**：右键 →「全部解压缩」→「解压」

> ⚠️ 记住这个**解压出来的文件夹**位置，第 5 步要用。不是 zip 本身，是解压后的文件夹。

## 第 3 步：打开扩展页面
Chrome 地址栏输入这串，按回车：
```
chrome://extensions/
```

## 第 4 步：打开开发者模式
页面**右上角**有个开关「Developer mode / 开发者模式」，点开（变蓝）。左上角会冒出几个按钮。

## 第 5 步：加载
点左上角 **「Load unpacked / 加载已解压的扩展程序」**，选第 2 步解压出来的**文件夹**，确定。

看到一张 "Norskvenn" 的卡片就是装好了 🎉

## 第 6 步（强烈建议）：固定图标
点 Chrome 工具栏右上角的**拼图块图标** 🧩 → 找到 Norskvenn → 点旁边的**图钉**。
这样 Norskvenn 图标就常驻工具栏了，后面调设置要用。

# 第二部分：基本使用

1. 打开任何一个挪威语 YouTube 视频（[NRK 挪威国家电视台](https://www.youtube.com/@NRK)有很多）
2. 在视频下方、Like/Share 按钮附近，找到红色按钮 **`🇳🇴 Live subs`**，点它
3. 等一下（本地模式第一次要下载 AI 模型，约 1-2 分钟；OpenAI 模式几秒）
4. 字幕出现在视频上：
   - **白字** = 挪威语
   - **黄字** = 英语翻译
5. **拖动字幕**可移到任意位置；**双击字幕**重置到底部
6. 再点一次按钮（这时显示 `⏹ Stop subs`）= 停止

## 重要：第一遍 vs 第二遍
- **第一遍看**：字幕是边听边生成的，所以会**滞后几秒到十几秒**——这是正常的，AI 不可能在话说出来之前就转好
- **第二遍看（或倒回去看）**：字幕已经存好了，**打开就有、完全同步、不花钱**

所以最佳用法：**让视频先放一遍（生成字幕），然后再正经看第二遍。**

# 第三部分：设置面板（点工具栏 Norskvenn 图标打开）

## ① 引擎选择（Engine）
- **Local Whisper**：免费，在你电脑上跑
- **OpenAI API**：要 API key，更准更快

### 怎么用 OpenAI 模式
1. 去 https://platform.openai.com/api-keys 注册并创建一个 API key（需要充值，最低 $5）
2. 复制那串 `sk-...` 开头的 key
3. 在设置面板选 **OpenAI API**，粘进去，点 **Save**
4. 回到 YouTube 刷新页面，重新点字幕按钮

> 💰 费用：约 $0.012/分钟视频（转写+翻译）。30 分钟视频约 $0.36。**同一个视频再看不再花钱**（用缓存）。
> 🔒 你的 key 只存在自己浏览器里，只发给 OpenAI，不发别处。

## ② 字幕更新频率（Subtitle update interval）
- **3s**：字幕来得最勤，但可能在词中间切断、稍不准
- **5s**：平衡（默认）
- **10s**：最准，但每 10 秒才来一批新字幕

> 想要**准** → 选 10s。想要**字幕勤快** → 选 3s。**费用都一样**（OpenAI 按音频时长收费，跟切多细无关）。

## ③ 字幕时间微调（Subtitle timing）
如果字幕跟人声对不齐：
- 字幕**慢了**（人说完了字幕才出）→ 点 **+0.1s / +0.5s**
- 字幕**早了**（字幕先于人声）→ 点 **−0.1s / −0.5s**
- **↺** = 归零

边看边调，立即生效，不用刷新。

## ④ 清空字幕缓存（Clear subtitle cache）
什么时候用：
- 某视频字幕时间老是对不上
- 升级了新版本，老字幕是旧版本生成的（可能不准）

点一下清空，然后重新转写一遍，字幕就是最新算法生成的。

# 常见问题 FAQ

**Q：每次重启 Chrome 弹警告"关闭开发者模式扩展程序"？**
点「保留 / Keep」。这是 Chrome 对所有非商店插件的提示，无害。

**Q：第一次点按钮卡在 "Loading model..." 很久？**
本地模式在下载 AI 模型（一次性，约 75MB）。下完就好，之后秒开。OpenAI 模式没这步。

**Q：状态条显示 "✓ cached, no API call" 是什么意思？**
表示这段字幕是从缓存读的，**没花钱**。看第二遍时会一直看到这个，正常。

**Q：字幕比视频内容慢？**
第一遍看是正常的（AI 来不及）。右上角会显示滞后几秒。**看第二遍就完全同步了。**

**Q：字幕内容是乱的 / 在瞎编？**
- OpenAI 模式有自检会自动丢掉低置信度的乱码段
- 如果还是乱：把"更新频率"调到 **10s**（短 chunk 容易把词切断导致乱猜）
- 音乐段、纯背景音可能没字幕，正常

**Q：挪威语字幕变成 "er det som er det som..." 重复？**
AI 在音乐/噪音段的幻觉，新版已经做了过滤。继续看，有人说话的段落会恢复。

**Q：我看的不是挪威语视频，字幕全乱？**
这插件**只为挪威语设计**。别的语言会被强行当挪威语解析，全乱。只在挪威语视频上用。

**Q：怎么更新到新版本？**
1. 重新去 [Releases 页](https://github.com/lp-capri-pl/norskvenn/releases/latest)下最新 zip
2. 解压
3. `chrome://extensions/` → Norskvenn 卡片 → 点 🔄 刷新图标 → 再点 "Load unpacked" 选新文件夹
4. 升级后建议：设置面板点一次 "Clear subtitle cache"
