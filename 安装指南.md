# Norskvenn 安装指南（5 分钟搞定）

这是一个 **Chrome 浏览器扩展**，能给任何挪威语 YouTube 视频生成 **挪威语 + 英语** 双语字幕，全部在你电脑本地跑，**不需要账号、不需要付钱**。

---

## 准备

你需要：
- 一台 Mac 或 Windows 电脑
- **Chrome 浏览器**（或 Edge / Brave / Arc 也行，都基于 Chromium）
  - 不知道有没有？[点这里下载 Chrome](https://www.google.com/chrome/)

---

## 第 1 步：下载

打开这个网址：

👉 **https://github.com/lp-capri-pl/norskvenn/releases/latest**

往下滚动，找到 **"Assets"** 那一栏，点 **`norskvenn-v0.1.0.zip`** 下载。

> 文件大概 10 MB，下载到你的"下载"文件夹（Downloads）里。

---

## 第 2 步：解压

到"下载"文件夹找到刚下的 `norskvenn-v0.1.0.zip`：

- **Mac**：双击它，会自动解压出一个文件夹（可能叫 `norskvenn-v0.1.0` 或 `dist`）
- **Windows**：右键 → "全部解压缩" → 点"解压"

解压完你会得到一个**文件夹**，里面有 `manifest.json` 等几个文件。

> ⚠️ **重要**：后面要选的是这个**解压出来的文件夹**，不是 .zip 文件本身。

---

## 第 3 步：打开 Chrome 的扩展程序页面

在 Chrome 地址栏（最上面那个输入网址的地方）输入：

```
chrome://extensions/
```

按回车。会打开一个页面，标题写着 "Extensions" 或 "扩展程序"。

---

## 第 4 步：开启"开发者模式"

页面**右上角**有一个开关，写着 **"Developer mode"** 或 **"开发者模式"**：

```
                                          [ Developer mode  ⚪ ]
                                                            ↑
                                                       点这里打开
```

点它，开关会变蓝。然后页面**左上角**会多出来三个按钮：

```
[ Load unpacked ]  [ Pack extension ]  [ Update ]
```

---

## 第 5 步：加载扩展

点 **"Load unpacked"**（"加载已解压的扩展程序"），会弹出文件选择窗口。

**选第 2 步解压出来的那个文件夹**（里面有 `manifest.json` 的那个），然后点"选择"或"打开"。

成功的话，页面上会出现一个卡片，写着：

```
Norskvenn — Norwegian YouTube subs   0.1.0
Generate bilingual Norwegian + English subtitles for any YouTube video...
ID: xxxxxxxxxxxxxxxx
```

🎉 装好了！

---

## 第 6 步：在 YouTube 上用

1. 打开任何一个 YouTube 视频（[挪威新闻 NRK 的频道](https://www.youtube.com/@NRK)有很多素材）
2. 在视频下方、标题旁边，找到 Like / Share 按钮**附近**，会有一个红色按钮：

   ```
   [ 🇳🇴 Live subs ]
   ```

3. 点它一下。
4. **第一次会下载 AI 模型（大概 75 MB），可能要等 30 秒 - 2 分钟**。状态条会显示进度，请耐心等待。
5. 模型加载完，开始播放视频，**字幕会出现在播放器上**：
   - **白色** = 挪威语原文
   - **黄色** = 英文翻译
6. **拖动字幕**可以移动到屏幕任意位置；**双击字幕**重置到底部居中。

## ❓ 常见问题

### Q1：每次重启 Chrome 都弹一个警告"关闭开发者模式扩展程序"
**答**：点 **"保留"**（Keep）就行。这是 Chrome 对所有非应用商店扩展的提示，无害，不用担心。

### Q2：第一次点按钮卡在 "Loading model..." 很久
**答**：正常，在下载 75 MB 的 AI 模型。下载速度看你网速，10 Mbps 大约 1 分钟。**只在第一次发生**，之后会被浏览器缓存，秒开。

### Q3：字幕出来了，但比视频内容慢很多（滞后好几秒）
**答**：右上角红条会显示 "subs lagging Xs behind playback"。这是因为 AI 在你电脑本地实时转写，**老一些的电脑跟不上视频速度**。
- 字幕仍然会显示（**变成淡灰色斜体**表示是"过期但仍然准确的"字幕）
- 解决办法：用配置更好的电脑，或者**暂停视频让 AI 追上**

### Q4：挪威语字幕变成奇怪的重复，比如 "er det som er det som er det som..."
**答**：这是 AI 在**音乐段或纯背景音**上的常见 bug（已经做了过滤但不一定全捕获）。继续看，等真正有说话的段落字幕会回来。

### Q5：我看的视频不是挪威语，字幕全是乱码
**答**：这个扩展**专为挪威语视频设计**。AI 被强制按挪威语解析，听到英语 / 中文等会乱猜。**只在挪威语视频上用**。

### Q6：怎么关掉字幕？
**答**：再点一次那个红按钮（这时它写着 "⏹ Stop subs"）。

### Q7：图标是个灰色拼图块（默认 Chrome 扩展图标），不好看
**答**：暂时还没做图标 😅。下个版本会加。功能正常用没影响。

## 出问题了找谁？

把 Chrome 扩展页面 (`chrome://extensions/`) 上 Norskvenn 卡片的 **"Errors"** 按钮里的红色错误截图发给 Capri，或者发邮件给项目作者。
